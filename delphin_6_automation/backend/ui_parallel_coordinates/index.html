What is this?

A multidimensional explorer of material data from the Delphin database.

The parallel coordinates displays the paramters of the material data in the database across multiple dimensions, colored by material categories.

If you have any question regarding the data above please contact: Thomas Perkov or Christian Kongsgaard

Controls

Brush: Drag vertically along an axis.
Remove Brush: Tap the axis background.
Reorder Axes: Drag a label horizontally.
Invert Axis: Tap an axis label.
Remove Axis: Drag axis label to the left edge.
